그래프 순회 (Graph Traversal) : DFS (Depth First Search) // BFS (Breadth First Search)

- preorder : 자신을 먼저 방문
- inorder : 자신을 가운데 방문
- postorder : 자신을 나중에 방문 


아래로 이동하면서 방문하지 않은 노드를 방문하며,
더 이상 아래 방향으로 방문하지 않는 노드가 없다면,
back track하여 그 지점에서 방문하지 않은 노드를 찾는다 
방문하지 않은 노드가 없다면 다시 back track하여 방문하지 않은 노드를 찾는다 

Code 1. 재귀방법 
 - 방문 중인 노드 
 - 방문한 노드  mark[v] = "visited"
 - 방문한 시간 pre[v] = current-time (방문할 때마다 1씩 증가) 
 - 부모 노드 parent[w] = v
 - DFS 완료된 시간 post[v] = current-time


Code 2. 비재귀 방법
 - stack을 이용 (Last In First Out)
